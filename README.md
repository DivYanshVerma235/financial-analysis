# Indian Stock Market Analysis- Reliance vs TCS

A financial data analysis project built using Python, comparing 
the performance of two major Indian stocks over a 1-year period.

# What this project covers
- Downloaded 1 year of real stock data using yfinance
- Normalized price comparison to fairly compare two stocks
- Daily returns calculation and distribution analysis
- Volatility comparison using standard deviation
- Correlation analysis between Reliance and TCS
- Sharpe Ratio calculation to measure risk-adjusted returns
- Maximum Drawdown analysis to measure downside risk

# Key Findings
- TCS showed higher volatility (std: 1.5%) vs Reliance (std: 1.27%)
- Both stocks had negative mean daily returns over the period
- Correlation of 0.21 — weak positive, good for diversification
- Negative Sharpe Ratio for both stocks indicates underperformance vs risk-free rate
- TCS had a larger maximum drawdown (-38.2%) vs Reliance (-20.6%)

# Tools Used
- Python, Pandas, Matplotlib, yfinance
- Jupyter Notebook
